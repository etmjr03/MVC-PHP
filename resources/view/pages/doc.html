Estrutura do projeto

app

  • Controller: É aqui que ficam todos os controladores (a ponte entre a view e os models)
  • Utils: É aqui que ficam armazenadas as classes utilitárias
  • Model: É aqui que ficam os modelos e entidades que cuidam dos dados e regras de negócio (representação de tabelas representadas por objetos, usuários, produto)

resources

responsável por guardar todos os conteúdos relacionados à layout (HTML, CSS, JS, imagens)

  • view: É aqui que são armazenados os layouts
